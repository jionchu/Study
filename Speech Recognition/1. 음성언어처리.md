# 음성언어처리

## 1. 음성언어처리 소개
음성대화는 인간과 인간 사이에서 가장 자연스러운 의사소통 수단이다.  
인간과 컴퓨터 사이에도 가장 자연스로운 상호작용 방법이다.  
사람이 자연스럽게 언어를 구사하듯 컴퓨터가 언어를 구사할 수 있도록 구현하는 것이 목표  

### 주요 분야
- 음성인식, 음성합성
- 자연언어 이해, 자연언어 생성
- 정보 검색, 질의 응답, 텍스트 요약
- 기계 번역, 자동 통역

### 음성언어처리
- 음성처리와 자연어처리가 융합된 분야

소규모 어휘 고립단어 음성인식이나 낮은 품질의 음성합성은 전기전자공학 분야의 음성신호처리 기술만 사용해도 구현할 수 있음  
하지만 입력음성을 음성신호로만 봐서는 연속음성인식이나 고품질 음성합성과 같은 난이도가 높고 필요성이 큰 테스크를 수행할 수 없다.
